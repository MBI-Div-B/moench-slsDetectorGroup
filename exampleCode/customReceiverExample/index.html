Custom SLS Receiver Example

Main Files

  • Makefile Edit to properly link the slsDetector libraries
  • dummyMain.cpp Main combining the required TCPIP interface and the custom dummy UDP implementation
  • dummyUDPInterface.h Dummy example of UDP interface implementation -- the UDP server never start listening!

Library Files

  • libSlsReceiver.so Receiver Library

Include Files

  • slsReceiverTCPIPInterface.h TCP/IP Interface between client and receiver
  • UDPInterface.h abstract UDP Interface of the receiver
  • UDPBaseImplementation.h Base Implementation of UDP Interface. Dummy Interface will be a child class of this
  • sls_receiver_defs.h Includes definitions used in the example files
  • sls_receiver_funcs.h Required by sls_receiver_defs.h
  • sls_receiver_funcs.h Required by sls_receiver_defs.h
  • MySocketTCP.h To create Sockets. Required by slsReceiverTCPIPInterface.h
  • genericSocket.h Required by MySocketTCP.h
  • receiver_defs.h Required by slsReceiverTCPIPInterface.h
  • utilities.h Required by UDPInterface.h
  • logger.h Required by UDPInterface.h
  • ansi.h Colored print. Required by most files

ZMQ Include Files

  • zmq.h Public Include file for ZMQ API users.
  • libzmq.a Static libary file
